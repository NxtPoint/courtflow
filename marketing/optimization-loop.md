# Marketing Optimization Loop — the standing discipline

> Marketing is never "done." It's a loop: **Measure → Learn → Adjust → Repeat.** Small, data-led
> tweaks compound. This doc is the standing cadence for NextPoint (and every future CourtFlow club).
> Owner: Tomo + Cowork (strategy) / Claude Code + Adspirer (execution).

## The one rule that governs everything
**Optimise toward the bottom of the funnel, not the top.**
Clicks → calls/enquiries → bookings → members/revenue. Always tune toward the metric closest to
money. Vanity metrics (impressions, raw clicks, cheap CPC) are diagnostics, never goals. This is *why*
conversion tracking is sacred — it's the sensor the whole loop runs on. **Bad signal in = bad
optimisation out.**

## The loop
1. **Measure** — pull the real numbers (Adspirer / Google Ads / GSC / Ahrefs / Klaviyo, plus
   CourtFlow's own `core.usage_event` tracking once live — **we use our own analytics, not GA4**):
   spend, CPC, CTR, search terms, conversions, and (once CourtFlow is live) bookings + revenue per channel.
2. **Learn** — where did money work, where did it leak? Which keywords/ads/audiences convert?
3. **Adjust** — change one or two things, not ten (so you can attribute the effect): add negatives,
   shift budget to winners, rewrite weak ads, fix a landing page, tighten/loosen geo.
4. **Repeat** — each pass is a marginal gain. ~2% better每 fortnight compounds hard over a year.

## Cadence
| Frequency | What | Who |
|---|---|---|
| **Weekly** (~15 min) | Search-terms prune → add negatives; glance at spend/CPC/conversions; pause obvious losers. | Claude Code + Adspirer (or Cowork) |
| **Monthly** (~1 hr) | Budget reallocation to winners; bid-strategy review (Max Clicks → Max Conversions/tCPA once data allows); refresh ad copy; review landing-page conversion; review geo + schedule. | Tomo + Cowork |
| **Quarterly** | Strategy step-back: channel mix (Google vs Meta), new campaign types, seasonal pushes (holiday camps, new-year fitness), competitor scan. | Tomo + Cowork |

## Decision triggers (let data pick the direction)
- **CPC too high / few clicks?** Improve Quality Score (ad↔keyword↔landing-page relevance) before raising bids; QS lowers CPC → more clicks per rand.
- **Clicks but no bookings?** It's a *funnel* problem (landing page / offer), not an ads problem. Fix the page, not the bid.
- **A keyword/ad group converts cheaply?** Give it more budget. **Converts expensively / not at all?** Cut or rework it.
- **Thinking of more budget?** Only after conversion value is known: if a R5 click books a R220/mo membership, scale; if not, fix the funnel first.
