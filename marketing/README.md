# NextPoint Tennis — Marketing

> Master index for everything marketing: paid ads, the optimization discipline, the tool stack, SEO, and
> social. Plain-English by design. Owner: Tomo + Cowork (strategy) / Claude Code + Adspirer (execution).

## The one philosophy
**Tools don't optimise — the loop does.** Every channel runs the same discipline: Measure → Learn →
Adjust → Repeat, always tuning toward the **bottom of the funnel** (bookings/members/revenue), never
vanity metrics. We own our funnel end-to-end (own analytics + own booking platform), which is the
compounding edge most clubs never build.
